入出力制御モジュール

入出力制御モジュールは下記機能を持つ。

  1. 初期化
  2. レジスタ読込み
  3. レジスタ書込み
▲上へ

初期化

デバイス初期化を行う。外部仕様(デバイス初期化機能)を参照のこと。

▲上へ

デバイス読込み

デバイス識別番号とレジスタをキーに下表のI/Oポートからレジスタを読込む機能を各モジュールに提供する。レジスタの読込みはMochiKernelライブラリのLibMkIoPortInByte()を用いる。


デバイス識別番号と読込み先I/Oポート番号
レジスタ デバイス識別番号別I/Oポート番号
0(COM1) 1(COM2)
RBR 0x03F8 0x02F8
IIR 0x03FA 0x02FA
LSR 0x03FD 0x02FD
▲上へ

デバイス書込み

デバイス識別番号とレジスタをキーに下表のI/Oポートにレジスタを書込む機能を各モジュールに提供する。IERとMCRへの書込みは指定されたbitだけを変更する機能を具備するため、一度該当レジスタを読込んだ後に設定値を反映させて書き込む。THRへの書込みは複数byteを連続して書き込む機能を具備する。レジスタの書込みはMochiKernelライブラリのLibMkIoPortInByte()を用いる。


デバイス識別番号と書込み先I/Oポート番号
レジスタ デバイス識別番号別I/Oポート番号
0(COM1) 1(COM2)
THR 0x03F8 0x02F8
IER 0x03F9 0x02F9
MCR 0x03FC 0x02FC
