バッファ管理モジュール

バッファ管理モジュールは下記機能を持つ。

  1. 初期化
  2. バッファ追加
  3. バッファ取出
  4. バッファクリア
▲上へ

初期化

バッファ管理モジュールは、デバイス毎に転送用バッファと受信用バッファの全4つのバッファをMLibライブラリのリングバッファを用いて管理する。初期化はMLibRingBufferInit()を用いる。設定パラメータを以下に示す。


MLibRingBufferInit()設定パラメータ
# 引数名 設定値
COM1 COM2
転送 受信 転送 受信
1 *pHandle リングバッファハンドルへのポインタ
2 entrySize 1
3 entryNum 4,096
4 *pErr エラー要因格納先ポインタ
▲上へ

バッファ追加

デバイス識別番号と種別(転送/受信)に対応するバッファにデータを指定サイズ分1byte毎に追加する機能を各モジュールに提供する。転送バッファへのデータ追加はMLibライブラリのMLibRingBufferPush()を用いる。受信バッファへのデータの追加はMLibRingBufferPushOW()を用いる（バッファフル時は最古のデータを破棄するため）。設定パラメータを以下に示す。


MLibRingBufferPush()およびMLibRingBufferPushOW()設定パラメータ
# 引数名 設定値
1 *pHandle リングバッファハンドルへのポインタ
2 *pData データへのポインタ
3 *pErr エラー要因格納先ポインタ
▲上へ

バッファ取出

デバイス識別番号と種別(転送/受信)に対応するバッファからデータを指定サイズ分1byte毎に取り出す機能を各モジュールに提供する。バッファからのデータ取出はMLibライブラリのMLibRingBufferPop()を用いる。設定パラメータを以下に示す。


MLibRingBufferPop()設定パラメータ
# 引数名 設定値
1 *pHandle リングバッファハンドルへのポインタ
2 *pData データへのポインタ
3 *pErr エラー要因格納先ポインタ

デバイス識別番号と種別(転送/受信)に対応するバッファをクリアする機能を各モジュールに提供する。バッファのクリアはMLibライブラリのMLibRingBufferClear()を用いる。設定パラメータを以下に示す。


MLibRingBufferClear()設定パラメータ
# 引数名 設定値
